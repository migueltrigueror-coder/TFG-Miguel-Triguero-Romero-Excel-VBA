Herramienta integral en Excel-VBA para la gestión de resultados académicos en el CUD
# 📊 Análisis Académico Automatizado (Excel-VBA)

> **Trabajo Fin de Grado - Ingeniería Mecánica (CUD)**
> Herramienta para la generación de informes estadísticos y patrones de rendimiento.

---

## 🚀 Funcionalidades Principales
* **Automatización:** Procesamiento instantáneo de actas con macros VBA.
* **Análisis Estadístico:** Cálculo de índices de dificultad, discriminación, asimetría y curtosis.
* **Informes Narrativos:** Generación automática de texto descriptivo para alumnos y exámenes.
* **Visualización:** Gráficos de cajas (boxplots) e histogramas dinámicos.

## 🛠️ Instalación y Uso
1. Descarga el archivo `.xlsm` de la carpeta raíz.
2. Habilita las macros en Microsoft Excel.
3. Introduce los datos en la hoja "Base de Datos" y pulsa el botón "Generar Informe".

## 📁 Estructura del Proyecto
* `src/`: Código fuente de los módulos VBA.
* `Anexos/`: Bases de datos de prueba (Alumnos Tipo y Exámenes Tipo).
